全局通用样式

根据设计组给出的SCOPA2.0统一视觉规范，整理并生成的前端样式文件，该全局样式文件会逐步完善。

目前可通过引用http://k1269.mlamp.co/static/scopa/theme/default/global.css实现。

通用字体
中文：思源黑体
链接

可用状态链接为元素添加.link-current类名。不可用状态链接为元素添加.link-current-disabled类名。

  • 链接

状态颜色

次级状态/临时状态为元素添加.status-minor类名。 安全状态/完成状态为元素添加.status-safe类名。 警示状态/失败状态为元素添加.status-warning类名。 锁定状态为元素添加.status-locking类名。

  • 次级状态/临时状态(色值：#fda738)
    (上升值域,停止中,已停止,重置中,更改密码中,创建中,更换系统中,解锁中)
  • 安全状态/完成状态(色值：#00c76c)
    (安全值域,运行中)
  • 警示状态/失败状态(色值：#ff474c)
    (启动失败,停止失败,重置失败,已过期,创建失败,密码错误,快照保存失败,到期失败)
  • 锁定状态(#aaaaaa)
    (已锁定)
<span class= "status-minor" > 次级状态/临时状态 </span> <span class= "status-safe" > 安全状态/完成状态 </span> <span class= "status-warning" > 警示状态/失败状态 </span> <span class= "status-locking" > 锁定状态 </span>
ICON状态颜色

运行中状态为元素添加.status-icon-running类名。 已停止状态为元素添加.status-icon-stop类名。 已过期状态为元素添加.status-icon-past类名。 已锁定状态为元素添加.status-icon-locking类名。

  • 运行中
  • 已停止
  • 已过期
  • 已锁定
<span class= "status-icon-running" > 运行中 </span> <span class= "status-icon-stop" > 已停止 </span> <span class= "status-icon-past" > 已过期 </span> <span class= "status-icon-locking" > 已锁定 </span>
LOADING状态

loading状态为元素添加.status-loading类名,为子元素添加.loading-icon类名。

  • 启动中
  • 停止中
  • 升级中
  • 创建中
<span class= "status-loading" > <i class= "loading-icon" ><i> 启动中 </span>
提示性状态

提示状态为元素添加.information-current类名。 疑问状态为元素添加.question-current类名。 警示状态为元素添加.warning-current类名。 正确状态为元素添加.right-current类名。 错误状态为元素添加.wrong-current类名。

<span class= "information-current" > </span> <span class= "question-current" > </span> <span class= "warning-current" > </span> <span class= "right-current" > </span> <span class= "wrong-current" > </span>
通用ICON

所有ICON圆角2px，线粗2px，大小为2X*2X，根据不同使用场景，相同ICON会有不同尺寸但线粗不变。

说明：点击以下字体所在的标签，即可复制字体类名

通用按钮
按钮在保证按钮文字辨识度的前提下,根据使用场景,尺寸可以调节。可通过修改按钮元素padding值修改按钮尺寸。
常规按钮

可用状态按钮为元素添加.btn-current类名。不可用状态按钮为元素添加.btn-current-disabled类名。

说明：常规可用按钮为#77aaff-#7fc7ff颜色渐变,hover状态下,弥散投影样式与按钮颜色相同为#77aaff-#7fc7ff颜色渐变,透明度为0.88。

  • 常规按钮
  • 常规按钮

警示按钮

警告按钮为元素添加.btn-current-warn类名。

说明：常规可用按钮为#ff4444—#fc6b6c颜色渐变,hover状态下,弥散投影样式与按钮颜色相同为#ff4444—#fc6b6c颜色渐变,透明度为0.88。警示按钮为特例，除非“警告”不建议使用此颜色按钮。
